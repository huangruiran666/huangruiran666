# huangruiran

AI-focused builder working on quantitative research tools, market-intelligence workflows, and compact data systems.

## Focus

- AI-assisted quant research and trading workflows
- Multi-source market signal collection and report generation
- ETL and modeling demos for financial and alternative-data analysis
- Desktop-first analyst tools with pragmatic cloud deployment helpers

## Featured Repositories

| Repository | What it does |
| --- | --- |
| [`quant-platform-windows`](https://github.com/huangruiran666/quant-platform-windows) | Desktop-first AI quant platform with market radar, strategy modules, and cloud helpers |
| [`TrendRadarAI`](https://github.com/huangruiran666/TrendRadarAI) | Market-intelligence workstation for collecting signals and generating daily reports |
| [`analysis-system`](https://github.com/huangruiran666/analysis-system) | Compact ETL and ML demo for alternative-data style financial analytics |
| [`quant`](https://github.com/huangruiran666/quant) | Single-file A-share quantitative research terminal |
| [`library`](https://github.com/huangruiran666/library) | Bilingual C++ command-line library management system |

## Stack

`Python` `pandas` `Flask` `Tkinter` `scikit-learn` `LightGBM` `GCP` `Azure OpenAI` `C++`

## Current Direction

Right now the focus is on making public repositories cleaner, more portable, and easier to understand at first glance, while keeping the projects practical instead of overdesigned.
